# Scripting
Automating tasks.

Double click the .cmd files to run them.

ip_address.cmd shows your ip address in a file named ip.txt

restart_in_5_sec.cmd restarts your pc in 5 seconds.

shutdown_in_5_sec.cmd shuts down your pc in 5 seconds.

sysinfo.cmd shows your system information in a file named sysinfo.txt
